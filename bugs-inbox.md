# HAD Game — Bugs Inbox

This is the bug intake queue for the bug-fix pipeline. New bugs are logged here with date, reporter, description, steps to reproduce, and priority. Issues are reviewed and triaged into the roadmap backlog or closed as invalid.

## Format

Each bug entry includes:
- **Date:** YYYY-MM-DD (reported)
- **Reporter:** Name or alias
- **Description:** What's broken or unexpected
- **Steps to reproduce:** How to trigger the bug
- **Priority:** Critical / High / Medium / Low
- **Status:** Pending / In Progress / Resolved
